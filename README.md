# Discover Dollar Assignment

Welcome to the Discover Dollar assignment! This document outlines the requirements and instructions for completing the assignment.

## Requirements

1. Understand the problem statement.
2. Implement a solution in your preferred programming language.
3. Write tests for your solution.
4. Submit your code via GitHub.

## Instructions

To get started, first clone the repository:

```bash
# Clone the repository
git clone https://github.com/{your_username}/Discover_dollar-assignment.git
```

Next, navigate to the project directory:

```bash
# Navigate to the project directory
dcd Discover_dollar-assignment
```

You can start implementing your solution right away.

## Testing

Make sure to write tests for your solution. You can do this by creating a new file `tests.py` in the root directory of the project:

```python
# Sample test
import unittest

class TestSolution(unittest.TestCase):
    def test_example(self):
        self.assertEqual(example_function(), expected_result)
```

Run your tests using:

```bash
# Run tests
python -m unittest tests.py
```

## Submission

Once you are satisfied with your solution and all tests are passing, you can submit your code:

1. Add your changes:

```bash
git add .
```

2. Commit your changes:

```bash
git commit -m "Your commit message"
```

3. Push your changes to GitHub:

```bash
git push origin main
```
